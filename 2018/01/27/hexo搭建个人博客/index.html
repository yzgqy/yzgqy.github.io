hexo+github搭建个人博客

一直以来都想搭建一个个人博客，在2017年初知道了用hexo搭建个人博客并搭建成功，但一直以来都没有写太多的文章。2018年的今天决定要重拾博客，搭建时又查阅了一番。

安装前提

在安装前必须保证电脑上安装了nodejs和git。

搭建网站

安装hexo

在安装了nodejs之后可以通过以下命令直接安装hexo

开启服务

1、清除缓存文件 (db.json) 和已生成的静态文件 (public)。

在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。

2、生产静态文件

可简写为

选项 描述
-d, –deploy 文件生成后立即部署网站
-w, –watch 监视文件变动

3、启动服务器

可简写为

选项 描述
-p, –port 重设端口
-s, –static 只使用静态文件
-l, –log 启动日记记录，使用覆盖记录格式

启动服务器。默认情况下，访问网址为： http://localhost:4000/

如果4000端口被占用，可以用ctrl+c停止服务器，使用“hexo server -p 端口号”命令来改变端口号

hexo与github page关联

关联前提

关联之前首先你得有个github账号，并与电脑的git设置好ssh配置，具体的配置可以见文章《git之SSH配置》

创建github仓库

在创建仓库时要注意Repository name的命名格式：

deploy:
type: git
repo: https://github.com/xxx/xxx.github.io.git
branch: master

npm instal lhexo-deployer-git –save

$ hexo clean

$ hexo g

$ hexo deploy

$ hexo d

$ hexo new [layout] <br>```<br>layout：post、page 和 draft

此时hexo的默认博客就已经部署好了，下面就可以进行文章的发布与个性化主题的设计了，可以参考hexo官方文档
